---
title: Mobile view
description: A read-only spectator view of a workspace on phones.
---

On a phone-sized screen, Typelets shows a streamlined spectator view instead of
the full desktop IDE. It is designed for following along, not for coding.

## What you can do on mobile

- **Problem** - read the interview prompt.
- **Files** - browse files and watch their contents update live as desktop
  collaborators edit (read-only - there is no code editor on mobile).
- **Preview** - view the workspace's running [preview](/hosting/persistent-preview/).
- **Chat** - read along, and send messages if you have an editing role.
- **Video** - join the call, when [video](/collaboration/video/) is enabled and
  your role allows it.

Interviewers can end an interview and owners can exit a sandbox from mobile.

## What you cannot do on mobile

- There is no code editor - you cannot edit files from a phone.
- There is no terminal.

For full editing and terminal access, use a desktop browser. Mobile is intended
for spectating, reviewing, and light participation while away from a computer.
