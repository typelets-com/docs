## What you can do with Typelets

<CardGrid stagger>
	<Card title="Host a static site" icon="rocket">
		Serve a real public site out of your workspace with a production static
		server, behind your own domain. See [Host a static site](/hosting/static-site/).
	</Card>
	<Card title="Keep an app always on" icon="setting">
		Turn on persistent preview and your container stays warm, so a web app,
		API, or bot keeps running. See [Persistent preview](/hosting/persistent-preview/).
	</Card>
	<Card title="Bring your own domain" icon="approve-check">
		Point a custom domain at a workspace via Cloudflare. See
		[Custom domains](/hosting/custom-domains/).
	</Card>
	<Card title="Automate with MCP" icon="puzzle">
		Drive workspaces and files from an MCP-capable client. See
		[MCP server](/automation/mcp/).
	</Card>
</CardGrid>
