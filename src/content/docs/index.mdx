## What Typelets does

<CardGrid>
	<Card title="Collaborate in real time" icon="pencil">
		Multi-file editing with live cursors (Yjs CRDTs), a shared terminal, and
		optional video - all in one workspace. See [Collaboration](/collaboration/editing/).
	</Card>
	<Card title="Run code safely" icon="rocket">
		Every workspace executes in its own isolated sandbox, with output
		streaming back live. See [Running code](/running-code/sandbox/).
	</Card>
	<Card title="Conduct interviews" icon="seti:notebook">
		Apply a problem, run a timed session, record it, and score against a
		rubric. See [Interviews](/interviews/overview/).
	</Card>
	<Card title="Host real apps & sites" icon="setting">
		Keep a workspace warm and serve a site or app at a stable URL, behind
		your own domain. See [Hosting](/hosting/persistent-preview/).
	</Card>
	<Card title="Automate with MCP & PATs" icon="puzzle">
		Drive workspaces and files from an MCP client or the REST API. See
		[Automation](/automation/mcp/).
	</Card>
	<Card title="No install" icon="laptop">
		Pure browser. Any platform, any device. Spectate from your phone.
		See [Mobile view](/collaboration/mobile/).
	</Card>
</CardGrid>
